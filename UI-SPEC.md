# 丘比 App — UI 设计规格说明（Figma 用）

> 整体风格：深色夜间系 · 玫瑰金渐变 · 毛玻璃卡片 · 轻奢感

---

## 全局设计规范

| 项目 | 值 |
|---|---|
| 主背景色 | `#07090F`（深夜蓝黑） |
| 主强调色 | 玫瑰红 `#F43F5E` |
| 辅助强调 | 琥珀金 `#EBC87A` |
| 卡片表面 | `rgba(17,20,30,0.68)` + 毛玻璃 blur |
| 主字色 | `#F6F1E8`（暖白） |
| 次要字色 | `rgba(255,255,255,0.45)` |
| 圆角 | 按钮 `12px`，卡片 `20px`，头像 `999px` |
| 字体 | 系统中文无衬线（SF Pro / PingFang） |

## 3. 开启匹配（流水线动画）

**路由：** `/match-start`

### 布局

```
┌─────────────────────────────────┐
│ 🔙  丘比 · 真实心动匹配         │
├─────────────────────────────────┤
│                                 │
│   「丘比会按心动设置双向筛选，   │
│    再在向量空间里找最合拍的人」  │  ← 说明文字
│                                 │
│           ┌───────┐             │
│    ✨光晕  │ 丘比  │  ✨光晕     │  ← 核心动画区
│           │ Logo  │             │    （旋转极光 + 呼吸圆圈）
│           └───────┘             │
│                                 │
│  ── 匹配进度 ─────────          │  ← 流水线进度（点击后出现）
│  ✅ 心动池扫描         500 人    │
│  ✅ 有效账号           238 人    │
│  ✅ 双向心动条件        42 人    │  ← 每步 520ms 依次出现
│  ✅ 向量相似 + 合拍     8 人     │    带扫描光条动画
│  🎯 双向心动阈值        2 人     │
│                                 │
│  ┌──────────────────────────┐   │
│  │    🌟 开启匹配            │   │  ← 主按钮
│  └──────────────────────────┘   │
│  ┌──────────────────────────┐   │
│  │    丘比小镇：发布找人      │   │  ← 次按钮（玫瑰描边）
│  └──────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

### 元素说明
- **动画核心**：圆形核心卡（Logo）+ 旋转极光光晕（conic-gradient）+ 呼吸发光效果
- **流水线步骤**：每步一张卡片，左侧标题+说明，右侧数字（人数）；逐步亮起
- **当前步**：扫描光条横扫动画（左→右 1.1s 循环）
- **成功态**：三颗心从两侧飞出碰撞（爆炸动画）→ 自动跳转匹配详情
- **失败态**：显示原因文字 + 「心动设置」按钮 + 「知道了」按钮

## 6. 真人聊天

**路由：** `/matches/[id]/chat`

### 布局

```
┌─────────────────────────────────┐
│ 🔙  [昵称]          [TA的主页] [打分] │  ← 顶部 Header
├─────────────────────────────────┤
│                                 │
│  ─── 昨天 14:32 ────────        │  ← 时间分隔线
│                          [我]   │
│           「你好～」  ◀ [气泡]   │  ← 自己的消息（右，玫瑰红）
│                        已读     │
│                                 │
│  [TA]                           │
│  [气泡] ▶ 「嗨！很高兴认识你」  │  ← 对方消息（左，毛玻璃）
│                                 │
│  ─── 今天 09:15 ────────        │
│                          [我]   │
│    「[图片缩略图 200×130px]」 ◀  │  ← 图片消息（点击放大）
│                        未读     │
│                                 │
│  [●●●]  对方正在输入…           │  ← 打字中气泡
│                                 │
├─────────────────────────────────┤
│  [表情] [____输入框__________] [📷][➤]│  ← 输入栏
├─────────────────────────────────┤
│  😊 😂 🥰 ❤️ 👍 ✨ 🌹 🎉 🔥 ⭐   │  ← Emoji 面板（展开态）
│  😎 🥺 😭 😡 🤔 💔 💕 👏 🙌 ✌️  │
│  ...（5行×10列，共50个）         │
└─────────────────────────────────┘
```

### 元素说明
- **背景**：深夜蓝黑 `#0A0D14`（可选渐变主题色）
- **自己气泡**（右）：玫瑰红渐变背景，右下角有气泡尖角，右侧头像「我」（金玫瑰渐变圆）
- **对方气泡**（左）：毛玻璃深色背景，左下角气泡尖角，左侧头像（昵称首字，玫瑰紫渐变圆）
- **时间分隔线**：居中横线 + 时间文字（灰色，小号）
- **已读/未读**：气泡下方小字「已读」（绿色）/ 「未读」（灰色）
- **打字中**：三个跳动圆点动画（左侧，带头像）
- **输入栏**：
  - 左：😊 表情按钮（点击展开/收起 Emoji 面板）
  - 中：Textarea（圆角，自动撑高，最大 3 行）
  - 右：📷 图片 + ➤ 发送按钮（玫瑰渐变圆角正方形）
- **图片预览行**：选图后在输入框上方显示缩略图 + 「移除」
- **Emoji 面板**：折叠收起，展开后 5行×10列，点击追加到输入框
- **图片 Lightbox**：点击图片消息 → 全屏黑背景 + 图片居中 + 点击关闭

## 设计注意事项（给 Figma 设计师）

1. **深色优先**：所有页面以深夜蓝黑为底，不设计浅色模式
2. **毛玻璃卡片**：`backdrop-blur: 12px` + `background: rgba(17,20,30,0.68)`
3. **气泡尾**：聊天气泡右下/左下有三角形尖角（CSS 三角，非图片）
4. **渐变色**：主要使用 Rose 500→600 玫瑰渐变，辅助使用琥珀金
5. **动画**：匹配流水线动画、气泡入场动画（从侧面 + scale 弹入）、打字三点动画
6. **安全区**：底部输入栏需适配 iPhone 底部安全区（`env(safe-area-inset-bottom)`）
7. **最大宽度**：内容区 max-width: 640px，居中显示（适配 PC 浏览器）
8. **字号**：主文字 14px，标题 16-18px，大强调数字 32-64px，辅助文字 11-12px
