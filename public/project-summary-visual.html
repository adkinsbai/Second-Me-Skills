丘比 · 各页面实时界面

如何使用：先在项目目录执行 npm run dev:3456，再用浏览器打开本页： http://127.0.0.1:3456/project-summary-visual.html （必须与 App 同源，iframe 才能加载各路由。）
首次打开时若未登录，会自动创建访客账号并刷新一次，以便小镇等页面不再被重定向到登录页 （需数据库可写；若访客失败，小镇 iframe 仍可能停在登录，可对照下方结构示意图）； 若你已手动登录，则不会创建访客。带 [id] 的页面请在底部「自定义路径」中粘贴完整地址。
